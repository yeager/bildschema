# Bildschema

Visual daily schedule with ARASAAC pictograms for children with autism.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install bildschema
```

### Fedora/RPM
```bash
sudo dnf install bildschema
```
