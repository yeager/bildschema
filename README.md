# Bildschema

Visual daily schedule with drag-and-drop and ARASAAC pictograms.

GTK4/Adwaita · Python · v0.3.3 · 78 translatable strings

Part of [Autismappar](https://autismappar.se) — free tools for children with autism, language disorders and ADHD.

## Install

### Ubuntu / Debian
```bash
sudo apt install bildschema
```
Repo: [yeager/debian-repo](https://github.com/yeager/debian-repo)

### Fedora
```bash
sudo dnf install bildschema
```
Repo: [yeager/rpm-repo](https://github.com/yeager/rpm-repo)

## Translation

Translated via [Transifex](https://app.transifex.com/danielnylander/bildschema/). 11 languages.
