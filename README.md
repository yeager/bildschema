# Bildschema

Visuellt dagsschema med drag-and-drop och ARASAAC-piktogram.

GTK4/Adwaita · Python · v0.3.3 · 78 översättningsbara strängar

Del av [Autismappar](https://autismappar.se) — fria verktyg för barn med autism, språkstörning och ADHD.

## Installation

### Ubuntu / Debian
```bash
sudo apt install bildschema
```
Repo: [yeager/debian-repo](https://github.com/yeager/debian-repo)

### Fedora
```bash
sudo dnf install bildschema
```
Repo: [yeager/rpm-repo](https://github.com/yeager/rpm-repo)

Översätts via [Transifex](https://app.transifex.com/danielnylander/bildschema/). 11 språk.
